Telco Customer Churn Prediction

A full-stack Machine Learning web app that predicts whether a customer will churn based on their account and service details.

**Live Demo:** https://telco-churn-s8uq.onrender.com

---

What I Did

* Built and trained multiple ML models:

  * Logistic Regression
  * Random Forest
  * XGBoost (final selected model)
* Compared performance and selected the best model
* Deployed a complete end-to-end system with a web interface

---

Features

* Real-time churn prediction
* Simple and interactive UI
* Fast API using Flask
* Fully deployed project

---

Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask (Python)
* **ML Models:** Logistic Regression, Random Forest, XGBoost
* **Deployment:** Render

---

Input Features

Gender, Payment Method, Internet Service, Contract, Tenure, Monthly Charges

---

How It Works

User input → Flask API → Data transformation → Model prediction → Result displayed
