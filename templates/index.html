How It Works
1
Fill in the form with customer demographics, services, and billing info.
2
Form POSTs JSON to your local Flask API at port 5000.
3
Transformer applies OneHotEncoding and numeric scaling via the saved pipeline.
4
XGBoost returns a binary prediction and churn probability score.
API Reference
POST http://127.0.0.1:5000/predict

Content-Type: application/json

Response:
 prediction: 0 | 1
 churn_probability: float
 status: "success"
